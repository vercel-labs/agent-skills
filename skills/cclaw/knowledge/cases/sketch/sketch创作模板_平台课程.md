## 一、模板结构说明

本文档为 Sketch 素描喜剧的**创作训练框架**，包含6个训练日模板。
每个训练日均有6个字段，创作时按顺序填写。

### 六大字段说明

| 字段 | 说明 |
|------|------|
| **作品标题** | 给你的作品起个名字 |
| **灵感** | 这个作品的灵感来源是什么？（生活观察/负面情绪/新闻事件/日常细节） |
| **前提** | 一句话概括整个 Sketch 的核心设定/情境/规则（Premise） |
| **结构** | 作品的叙事结构：如何建立→如何推进→如何收尾 |
| **全文** | 完整的 Sketch 脚本文本 |
| **教练反馈** | 创作完成后的复盘/优化点 |

## 三、训练路径总结

| 训练日 | 任务 | 关键词 |
|--------|------|--------|
| Day 6 | 完成第1个原创作品 | 从零到一 |
| Day 7 | 优化第1个原创作品 | 打磨 |
| Day 9 | 完成第2个原创作品 | 尝试不同角度 |
| Day 10 | 优化第2个原创作品 | 迭代 |
| Day 11 | 24小时限时创作第3个作品 | 速度 |
| Day 12 | 24小时限时创作第4个作品 | 效率复盘 |

### 创作核心流程（可复用）
1. **锁定灵感** — 观察生活，记录触动点
2. **提炼前提** — 用一句话说清楚设定
3. **设计结构** — 建立→推进→收尾
4. **写出全文** — 对话+动作+节奏
5. **获取反馈** — 讲给别人听，记录反应
6. **优化迭代** — 哪里不好笑改哪里

## 四、Sketch（素描喜剧）核心特点

### 与脱口秀的区别

| 维度 | 脱口秀 | Sketch素描喜剧 |
|------|--------|----------------|
| 人物 | 单人（我） | 多个角色 |
| 时长 | 5-15分钟 | 1-5分钟 |
| 结构 | 铺垫+笑点线性推进 | 情境建立+反复强化+意外收尾 |
| 笑点密度 | 分散，多段子串联 | 高密度，同一情境反复炸 |
| 核心 | "错误"艺术 | "情境"艺术 |

### Sketch 的前提（Premise）类型
- **情境型**：两个人被困在电梯里，但电梯只是不动，并没有坏
- **身份型**：如果医生比病人还紧张
- **规则型**：如果所有人说话都只能反着说
- **反转型**：前面一切正常，最后一个细节颠覆全局
