## 二、什么是脱口秀

### 核心定义
**脱口秀是错误的艺术**

### 基本参数
| 参数 | 内容 |
|------|------|
| 时长 | 5分钟（线上），线下15-20分钟 |
| 人物 | 我 |
| 节奏 | 聊天 |
| 共鸣 | 不是逗观众笑，是观众觉得好笑 |

### 段子结构
> **铺垫 + 一个错误引发的笑点**

#### 例子（健身教练段子）：
铺垫：现在很多人都办卡，有些人办卡是为了增肌，有些人办卡是为了减脂，有些人办卡是因为**一时糊涂**

错误：男人深蹲，女生受不了，女人深蹲，男人受不了，**但我们的学员不一样，他们深蹲，腿受不了**

结论：在教练严格指导下，学员完成了从身体瘦弱到肢体残疾的转变

## 三、铺垫是灵魂

### 铺垫怎么写？

**核心方法：负面情绪共鸣**

### 负面情绪的四大类型：奇、难、怕、蠢

| 类型 | 公式 |
|------|------|
| 奇（奇怪） | 我觉得XXX太奇怪了，因为YYY |
| 难（困难） | 我觉得XXX太困难了，因为YYY |
| 怕（害怕） | 我觉得XXX太可怕了，因为YYY |
| 蠢（愚蠢） | 我觉得XXX太蠢了，因为YYY |

### 负面情绪素材来源
- 健身房推销办卡
- 办卡后不好好教学
- **生活中负面情绪就是我们的素材**

### 负面情绪转化公式
**我觉得（____）太（奇/难/怕/蠢）了，因为（____）**

#### 例句：
- "我觉得减肥太困难了，因为一减肥就想吃东西"
- "我觉得我爸太奇怪了，因为他干什么都要计算一下"
- "我觉得婆媳矛盾太可怕了，因为我实在不知道该帮谁"
