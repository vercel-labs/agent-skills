# Evaluation Queries: Web Application Verification

## Should Trigger (8)

1. "Scan this website for security vulnerabilities: https://example.com"
2. "Check this dApp frontend for XSS vulnerabilities."
3. "Verify this web application for insecure APIs."
4. "Does this website have OWASP Top 10 issues?"
5. "Audit this frontend before launch."
6. "Is this dApp interface secure?"
7. "Scan this website for common web vulnerabilities."
8. "Check this URL for frontend security flaws: https://my-dapp.com"

## Should Not Trigger (9)

1. "How do I build a website?" (Development guide)
2. "What's OWASP?" (General explanation)
3. "How do I use a web framework?" (Framework guide)
4. "What's an API?" (API education)
5. "How do I host a website?" (Hosting guide)
6. "Explain web server security." (Education)
7. "What's a reverse proxy?" (Architecture info)
8. "How do I implement HTTPS?" (SSL/TLS guide)
9. "What's DNS?" (DNS education)
