# Evaluation Queries: OpenClaw AI Agent Verification

## Should Trigger (8)

1. "I built an AI agent with a file upload tool. Is it secure?"
2. "Audit this agent configuration for vulnerabilities."
3. "Can you verify if my agent has permission issues?"
4. "Check if my agent is vulnerable to prompt injection."
5. "Is this agent safe to deploy in production?"
6. "Review my agent's message handling for security issues."
7. "I want to gate-check agents before they execute. How secure is this config?"
8. "Validate this agent's tool access permissions."

## Should Not Trigger (9)

1. "How do I build an AI agent?" (General guidance, not verification)
2. "What's the latest AI model?" (Model info)
3. "Help me debug my agent's output." (Debugging assistance)
4. "What's machine learning?" (General education)
5. "Can you improve my agent's performance?" (Performance optimization)
6. "How do I deploy an agent?" (Deployment guide)
7. "What's the difference between LLMs?" (Model comparison)
8. "Write an agent for me." (Development request)
9. "Explain AI safety." (General education)
